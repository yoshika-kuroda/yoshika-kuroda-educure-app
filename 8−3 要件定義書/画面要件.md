# 画面要件定義

## 画面要件一覧（マトリクス表）

| ID | 画面名 | 画面説明 | 遷移画面 | 備考 |
|----|--------|----------|----------|------|
| S-01 | トップ画面 | アプリのタイトルとログインボタンを表示する | ログイン画面 | |
| S-02 | ログイン画面 | Googleアカウントでログインする | トップ画面、単語一覧画面 | Google OAuth 2.0 |
| S-03 | 単語一覧画面 | 登録した単語を一覧で表示する | 単語詳細画面、新規登録画面 | |
| S-04 | 単語詳細画面 | 単語の詳しい情報を表示する | 単語一覧画面、編集画面 |単語削除可能 |
| S-05 | 単語登録画面 | 新しい単語を登録する | 単語一覧画面 | CSV登録対応 |
| S-06 | 単語編集画面 | 単語の内容を修正する | 単語詳細画面 | |
| S-07 | 暗記モード画面 | 単語を覚えるための画面 | 単語表示画面 | |
| S-08 | カテゴリ管理画面 | カテゴリの管理を行う | 単語一覧画面 | カテゴリ追加、編集可能 |
| S-09 | CSVインポート画面 | CSVファイルを読み込む | 単語一覧画面 | API不使用 |
| S-10 | CSVエクスポート画面 | 単語をCSVファイルで保存する | 単語一覧画面 | API不使用 |
