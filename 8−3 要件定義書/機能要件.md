# 機能要件定義

## 機能要件一覧（マトリクス表）

| ID | 機能名 | 機能詳細 | 関連データ | 関連ユーザー | 備考 |
|----|--------|----------|------------|--------------|------|
| F-01 | ログイン | Googleアカウントを使ってログインする | ユーザーID、GoogleユーザーID、メールアドレス、名前 | 一般ユーザー、管理者 | Google OAuth 2.0（無料API） |
| F-02 | ログアウト | ログイン状態を終了する | ログイン情報 | 一般ユーザー、管理者 | |
| F-03 | ユーザー情報削除 | ユーザー情報を削除 | ユーザーID、名前、メールアドレス、ユーザー区分| 管理者 | |
| F-04 | 単語登録 | 単語と意味、カテゴリを登録する | 単語、意味、カテゴリ | 一般ユーザー | |
| F-05 | 単語一覧表示 | 登録した単語を一覧で表示する | 単語、意味（一部） | 一般ユーザー、管理者(閲覧のみ) | |
| F-06 | 単語詳細表示 | 単語の詳しい情報を表示する | 単語、意味、カテゴリ、登録日、更新日 | 一般ユーザー、管理者(閲覧のみ) | |
| F-07 | 単語編集 | 登録した単語を修正する | 単語、意味、カテゴリ | 一般ユーザー | |
| F-08 | 単語削除 | 登録した単語を削除する | 単語ID | 一般ユーザー | |
| F-09 | 単語検索 | 入力した文字を含む単語を検索する | 単語 | 一般ユーザー、管理者(閲覧のみ) | |
| F-10 | カテゴリ検索 | カテゴリごとに単語を表示する | カテゴリ | 一般ユーザー、管理者(閲覧のみ) | |
| F-11 | 並び替え | 単語を五十音順や日付順で並び替える | 単語、登録日 | 一般ユーザー、管理者(閲覧のみ) | |
| F-12 | 暗記モード | 単語を覚えるための学習モードを使う | 単語、意味 | 一般ユーザー | |
| F-13 | CSVインポート | CSVファイルから単語をまとめて登録する | 単語、意味、カテゴリ | 一般ユーザー | API不使用 |
| F-14 | CSVエクスポート | 単語データをCSVで保存する | 単語一覧 | 一般ユーザー | API不使用 |
| F-15 | カテゴリ一覧表示 | 登録したカテゴリを一覧で表示する | カテゴリ名 | 一般ユーザー、管理者(閲覧のみ) | |
| F-16 | カテゴリ詳細表示 | 登録したカテゴリの詳細を表示 | カテゴリ、登録日、更新日 | 一般ユーザー、管理者(閲覧のみ) | |
| F-17 | カテゴリの追加 | 新規カテゴリを登録する | カテゴリ名 | 一般ユーザー | |
| F-18 | カテゴリの編集 | 登録したカテゴリを編集する | カテゴリ名 | 一般ユーザー | |
| F-19 | カテゴリの削除 | 登録したカテゴリを削除する | カテゴリID | 一般ユーザー | |
