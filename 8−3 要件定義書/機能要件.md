# 機能要件定義

## 機能要件一覧（マトリクス表）

| ID | 機能名 | 機能詳細 | 関連データ | 関連ユーザー | 備考 |
|----|--------|----------|------------|--------------|------|
| F-01 | ログイン | Googleアカウントを使ってログインする | ユーザーID、GoogleユーザーID、メールアドレス、名前 | 一般ユーザー、管理者 | Google OAuth 2.0（無料API） |
| F-02 | ログアウト | ログイン状態を終了する | ログイン情報 | 一般ユーザー、管理者 | |
| F-03 |単語登録 | 単語と意味、カテゴリを登録する | 単語、意味、カテゴリ | 一般ユーザー、管理者 | |
| F-04 | 単語一覧表示 | 登録した単語を一覧で表示する | 単語、意味（一部） | 一般ユーザー、管理者 | |
| F-05 | 単語詳細表示 | 単語の詳しい情報を表示する | 単語、意味、カテゴリ、登録日、更新日 | 一般ユーザー、管理者 | |
| F-06 | 単語編集 | 登録した単語を修正する | 単語、意味、カテゴリ | 一般ユーザー、管理者 | |
| F-07 | 単語削除 | 登録した単語を削除する | 単語ID | 一般ユーザー、管理者 | |
| F-08 | 単語検索 | 入力した文字を含む単語を検索する | 単語 | 一般ユーザー、管理者 | |
| F-09 | カテゴリ検索 | カテゴリごとに単語を表示する | カテゴリ | 一般ユーザー、管理者 | |
| F-10 |  並び替え | 単語を五十音順や日付順で並び替える | 単語、登録日 | 一般ユーザー、管理者 | |
| F-11 |　暗記モード | 単語を覚えるための学習モードを使う | 単語、意味 | 一般ユーザー | |
| F-12 | CSVインポート | CSVファイルから単語をまとめて登録する | 単語、意味、カテゴリ | 一般ユーザー、管理者 | API不使用 |
| F-13 | CSVエクスポート | 単語データをCSVで保存する | 単語一覧 | 一般ユーザー、管理者 | API不使用 |
| F-14 | カテゴリ管理 | カテゴリの追加・編集・削除を行う | カテゴリ名 | 一般ユーザー、管理者 | |
